# Cinematic UI Rules

## Motion

- All KPI counters animate upward
- Panels fade + slide in
- Hover glow on cards
- Map transitions smooth
- No abrupt value changes

---

## Effects

- Subtle neon glow on important numbers
- Pulsing red for critical risk
- Animated sparkline in KPI cards
- Background animated grid

---

## Typography

- Bold for KPI numbers
- Uppercase micro labels
- Generous spacing
- Minimal text clutter

---

## Layout Principles

- One command centre screen
- No scrolling if possible
- Responsive grid
- Clean separation of panels
