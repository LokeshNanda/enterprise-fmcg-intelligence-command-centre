# Enterprise FMCG Intelligence Command Centre

## Vision

Build a cinematic, futuristic enterprise dashboard for a large FMCG conglomerate.

The experience should feel like:
Bloomberg Terminal × Palantir Gotham × Tesla UI

Single command-centre screen.
Dark theme.
Smooth motion.
Animated KPIs.
Realistic mock enterprise data.

Deployable on Vercel using Next.js.

---

## Core Experience Goals

- Executive-level clarity
- Realistic enterprise KPIs
- Smooth transitions
- Subtle glowing accents
- Animated counters
- Live mock data refresh
- AI-generated executive insight panel

---

## Tech Stack

- Next.js (App Router)
- TailwindCSS
- Framer Motion
- Recharts
- Mapbox GL
- TypeScript
- Server-side mock data generator

---

## Layout Structure

Top Section:
- KPI Bar (6 animated KPIs)

Center:
- India Heatmap with revenue/risk toggle
- Hover interactions
- Right-side floating drill-down drawer

Left Panel:
- Manufacturing Intelligence

Right Panel:
- Sales & Distribution Momentum

Bottom Left:
- Finance & Liquidity Panel

Bottom Right:
- AI Executive Insights Panel
